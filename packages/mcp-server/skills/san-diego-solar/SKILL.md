# San Diego Solar/PV Permit Skill

## Purpose
Guide solar panel and battery storage permit applications in San Diego.

## Permit Types
- **Electrical PV Combo**: Solar + electrical work (~9K issued)
- **Photovoltaic SB 379**: Streamlined solar per SB 379 (~5K issued)
- **Standard Electrical Permit**: For electrical-only modifications

## State Streamlining (AB 2188)
- Cities must provide streamlined process for small residential solar
- Electronic submittal required
- Must use checklist-based review
- 1-3 week processing typical

## Plan Requirements
1. Site plan showing panel locations
2. Electrical single-line diagram
3. Equipment specifications (panels, inverter, racking)
4. Structural attachment details
5. Fire setback compliance (3-foot ridge setback, access pathways)
6. Utility interconnection application

## Battery Storage Additions
- Fire code compliance required
- Indoor installations need ventilation assessment
- Outdoor installations need clearance from doors/windows
- ESS (Energy Storage System) label required
- May trigger additional fire department review

## Common Project Scenarios
- **Rooftop residential (most common)**: PV permit, 1-3 weeks
- **Ground-mounted residential**: PV permit + possible building permit for structure
- **Commercial rooftop**: Building permit required, 4-6 weeks
- **Solar + EV charger**: May combine into single permit
- **Solar + battery**: Additional electrical and fire review

## Tips
- Expedited review available and recommended
- Title 24 compliance is automatic for solar (reduces energy budget)
- Net metering with SDG&E requires separate utility application
- Consider Rapid Review for fastest turnaround (50% surcharge)
