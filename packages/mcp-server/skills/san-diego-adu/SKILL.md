# San Diego ADU Permit Skill

## Purpose
Provide detailed guidance for Accessory Dwelling Unit (ADU) and Junior ADU (JADU) permit applications in San Diego.

## ADU Types

### Detached ADU (new construction)
- Max size: 1,200 sq ft
- Max height: 16 feet
- Setbacks: 4 feet (side and rear)
- Parking: None required within 1/2 mile of transit

### Attached ADU (addition to primary)
- Max size: 50% of existing living area or 1,200 sq ft (whichever is less)
- Must comply with primary structure setbacks
- Attached to or within the existing primary dwelling

### Garage Conversion ADU
- Convert existing garage to living space
- No setback changes required (existing structure)
- May need parking replacement if not near transit
- Often qualifies for streamlined review

### Junior ADU (JADU)
- Max size: 500 sq ft
- Must be within existing primary dwelling footprint
- Requires interior connection to primary home
- Efficiency kitchen allowed (no gas, small appliances)
- Owner occupancy required for JADU (not standard ADU)

## State Law Requirements (Gov. Code §66310-66342)
- Ministerial (non-discretionary) approval
- 60-day processing deadline
- No impact fees for units under 750 sq ft
- No owner-occupancy requirement (except JADU)
- One ADU + one JADU allowed per single-family lot
- Cannot be sold separately (exceptions for affordable housing)

## Required Forms
- DS-345: Application for Building Permit
- DS-530: ADU Supplemental Application
- DS-560: Plan Submittal Requirements

## Common Rejection Reasons
1. Incomplete application (missing DS-530)
2. Plans missing required details (site plan, floor plan, elevations)
3. Exceeding size or height limits
4. Setback violations
5. Missing Title 24 energy calculations
6. Insufficient parking analysis (if required)

## Tips for Success
- Pre-application consultation with DSD is highly recommended
- Include CALGreen compliance documentation
- Show all existing structures on site plan
- Include utility connection details
- Fire sprinklers may be required if primary dwelling has them
